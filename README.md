# ForgeNote

Aplicação web em Python para organizar personagens, ideias e histórias em um só lugar.

## Tecnologias
- Python
- Flask
- HTML
- CSS

## Funcionalidades atuais
- Tela de login
- Validação com fake database
- Home inicial

## Como rodar
new terminal
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python run.py
